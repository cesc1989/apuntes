# Shlink - Self Hosted link Shortener

Web: https://shlink.io/

Armo este documento porque me encontré con este proyecto y vi cosas interesantes que podría llevar a Enlacito. Las pondré aquí.

## API 🌟

Algo recurrente en muchos de estos servicios es proveer una API. Lo hacen porque son ideales para campañas de mercadeo o para integrar con plataformas de gestión de redes sociales como Buffer.

Anoto para darle importancia a esta característica que no existe en Enlacito.

## Acceso limitado

Como vi en otros servicios, Shlink también deja reducir el acceso a los links mediante fechas de vencimiento o cantidad de visitas.

## Detección de Bots 🔑

Esto ya está en Enlacito. Llama la atención que lo mencionan. Pensé que era algo que todos hacen pero tal vez no.

Podría ser algo a destacar en alguna parte de Enlacito. Sea la landing o como info al ver las estadísticas de los enlaces.
