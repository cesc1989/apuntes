Como desarrolladores no todo lo podemos resolver escribiendo código. Algunas veces hay cierto tipo de problema que se resuelve con una herramienta externa. Y no hablo de integraciones con APIs de terceros. Hablo de cosas sencillas como pasar un contenido de un tipo a otro.

Un mal ejemplo sería pasar un documento PDF a Word. ¿Por qué haríamos eso? No sé pero a veces toca ese tipo de cosas. A veces toca comparar dos JSON. O de pronto poner texto Lorem Ipsum en una UI que estamos creando.

Como decía antes. Cosas que no se pueden resolver a punta de código.

Es para esos casos que traigo tres herramientas que son un conjunto de otras. Son las llamadas Dev Tools.

## Dev Tools: Una Caja de Herramientas

Tal como un mecánico, soldador, albañil, carpintero, etc, tiene su caja de herramientas con martillos, cinceles, destornilladores, llaves inglesas, llaves, entre otras. De esa misma forma es como funcionan las Dev Tools que vengo a compartir. Son tres.

- Mi favorita y más completa: <a href="https://it-tools.tech/" target="_blank">IT Tools</a>
- Una bastante completa y variada: <a href="https://omnitools.app/" target="_blank">Omni Tools</a>
- De último pero no menospreciada: <a href="https://devtoys.app/" target="_blank">Dev Toys</a>

Todas ellas ofrecen una gran variedad de _herramientas_ para completar tareas puntuales.

Ejemplo de las cosas que podemos hacer con IT Tools:

- Convertir de JSON a TOML, YAML, XML y vice versa
- Formatear YAML, XML o JSON
- Generar texto Lorem Ipsum
- Comparar dos textos similares (diff)
- Generar UUIDs

Omni Tools hace varias de estas y va más allá porque ofrece gestión y conversión de archivos PDF (alternativa a iLovePDF que respeta la privacidad), conversión de vídeos e imágenes, y más.

---

Me parecen muy bacanos estos proyectos, además de útiles, así que quise compartirlos por este espacio.

Espero sean de utilidad.
